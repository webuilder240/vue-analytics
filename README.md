# vue-analytics

Resource plugin for Vue.js.

## Setup

### Webpack/Browserify

Add `vue` and `vue-analytics` to your `package.json`, then `npm install`, then add these lines in your code:

```js
var Vue = require('vue');

Vue.use(require('vue-analytics'));
```
